ELECTRIBESX.COM
I got tired of the lack of resources for Electribe SX owners and decided to make one. It's ugly and basic, as I always put utility over presentation. Please contribute.
Software Tools
Open Electribe Editor
Formerly known as the ESX Wave Organizer, this is the big daddy of ESX editors. Author Jeff did a ton of work reverse engineering the file format so we could all benefit, and the other tools on this page use his work to some extent. All ESX owners owe him a debt of gratitude.
Node Electribe File Format Toolkit
An NPM package which allows parsing, editing, and writing the ESX file format.
Electribe Manager command line tool
Java command line tool for ESX file management, written by the webmaster. If you want to write properties files to build your ESX sample set, use this.
